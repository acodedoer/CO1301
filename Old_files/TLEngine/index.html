Getting Started with the TL-Engine

The ‘TL-Engine’ is a special Game Engine to help teach games programming. It has been developed at the University of Central Lancashire specifically for teaching games programming. The TL-Engine has been written by Laurent Noel. Essentially it is a library of functions which can be called from within a C++ program. In this module, you will be writing C++ programs which call the TL-Engine to make 3D games. We will be using the latest version of the TL-Engine (V2.3 beta 6) with Visual Studio 2019 for this module. These tools are installed on the Games Lab machines on campus, however while working from home you need to install them on your personal machines.

Installing the Tl-Engine

  1. You need to first download and install the DirectX SDK from https://www.microsoft.com/en-gb/download/details.aspx?id=6812

  2. You also need to install Visual Studio 2019 from https://visualstudio.microsoft.com/downloads/

  3. Finally, download and install the latest version of the TL-Engine from Gamesnorthwest.net.

Creating a TL-Engine Project

  1. First make a new folder for your work on the desktop and give it a unique name so you can identify it easily.

  2. At the end of each session in the games lab you must save your work to a pen drive or a copy it to your account via "Remote Access". Do not assume that your work will be safe on the computer for later.

  3. Furthermore, since your work is stored locally on the hard-drive of the machine, anything you leave there is accessible by anyone else who logs onto the machine. BE CAREFUL not to leave assignment work there for others to steal!

  4. Navigate to the TL-Engine through the start menu, or search “TL-Engine” in the taskbar search box

    HINT

  5. ‘Create New TL-Engine Project’ should be visible, click on it

    HINT

  6. A TL-Engine dialog box will appear, type a descriptive name for your project in the ‘Name’ edit box and browse to the folder you made earlier in the ‘Location’ edit box. Visual Studio 2019 is selected by default.

    HINT

  7. Click OK to create your project.

    HINT

Running the Project

  1. The TL-Engine wizard creates a fully working Visual Studio project to use a 3D engine. You can see the project structure in the “Solution Explorer” window on the left.

    HINT

  2. Expand the project structure by clicking on the arrow-head icon to the left of the project name in the solution explorer pane.

    HINT

  3. View the program code by double-clicking the on the file ending in ‘.cpp’ (this is a C++ source file).

    HINT

  4. You can run the program straight away by clicking the green coloured play icon near the menus at the top of the screen (or press F5)

    HINT

  5. The program will compile and run…

    HINT

  6. First a console window appears and tells you what is happening

    HINT

  7. Then a second blank grey window appears - this second window is a 3D engine showing an empty scene

    HINT

  8. Press ‘Alt’+F4 to close this window, and return to the code

    HINT
