3. An Array of Spheres

  1. Remember arrays? You have done arrays in Week 6 in the programming module and also in this module. An array is a way of referring to a group of variables by a single name. For example the code below will create an array of 10 IModels. 

    IModel* spheres[10];

  2. We can refer to the group of variables in an array by just using the array name. The individual variables can be referenced using the array index, a number inside square brackets. The array index always starts at 0, this means that the last element of the array is one less than the size of the array (the last element of an array of size 10 is array index 9). The code below references the last element of the IModel* array given above: 

    spheres[9];

    Going over the edge of an array causes a run-time error.

4. Sine Wave

  1. The next step is to create a sine wave using the line of spheres.

  2. At the moment as the spheres move across the screen only their x coordinates are being changed. The y coordinate of the spheres need to be changed in order to produce a sine wave. Therefore, you need to set the y coordinate of each sphere to sin(x), where x is simply the current x coordinate of the sphere. You will need to:

    1. get the current x coordinate of the sphere
    2. apply sin() in order to obtain the new y value
    3. set the y coordinate of the sphere to this new value

    In order to use the sin() function you need to include the "maths.h" library in your program.
