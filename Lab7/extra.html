CO1301: Games Concepts
Lab 7 - Drone Wars

1. Extending the Drone Program

  1. If you have completed the main lab tasks, here are a few ideas that you can use to extend your program:

    • Test if the bullets have hit playerDrone (using vector distances).
    • Have enemyDrone shoot more than one bullet at a time.
    • Add more enemy drones.
    • Keep track of the playerDrone's health using some visible UI element (e.g. percentage of health left) and end the game if the playerDrone has no health left
