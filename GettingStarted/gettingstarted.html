Lab 0: Installing TL-Engine on YOUR PC

1. Downloading and Installing Required Software

The instructions that following will guide you through downloading and installing the TL-Engine and all other required software on your PC. If you are using PCs in the Games Lab, then you do not have to install any new software, the TL-Engine is available on all computers in the Lab.

  1. Start by downloading (from here) and installing Visual Studio 2019, make sure you also choose to install the C++ Workload.

  2. Then, download (from here) and install the DirectX SDK.

  3. Next, download (from here) and install the latest version of the TL-Engine.

  4. You have now installed all the required tools.

3. Creating a TL-Engine Project: Instructions

  1. Click on the start menu and search for TL-Engine.

  2. The TL-Engine template wizard app should appear, click on it to start the TL-Engine template wizard. You can also access this through 'All Programs'->'TL-Engine'->'Create New TL-Engine Project…'

  3. In the name field, provide a name for your project.

  4. In the location field, provide a location for your project to be saved.

  5. The template wizard should by default have Visual Studio 2019 and "Automatically open new project" selected.

  6. Click OK. The project will be created and started in Visual Studio 2019.

  7. Once Visual Studio has opened your project, you should be able to see a "Solution Explorer" window to the right of your screen. This window contains the name of your project, with a tiny right facing icon by its left side, click on the icon. You should now be able to see your project structure.

  8. Look for a file that has a name ending in ‘.cpp’ (this is a C++ source file) and click to open it.

  9. The file contains a template code generated by the TL-Engine, it does not do much, and you will learn to add to it to create more interesting programs in the labs. However, you can still run the project by pressing F5 or clicking the green 'play' button on the top menu.

  10. The project will compile and run... a console window appears and tells you what is happening, then a second blank grey window appears - this second window is a 3D engine showing an empty scene.

    Press ‘Alt’+F4 to close this window, and return to the code
