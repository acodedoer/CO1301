CO1301: Games Concepts
Lab 2 - Round a Square Path

1. Basic Square Path

  1. Create a new TL-Engine project called Lab2_SquarePath_Project.

  2. Load a floor mesh using "floor.x", then create a floor model at the origin.

  3. Load a sphere mesh, then create a sphere model at position (-75, 10, 75), then follow the instcutions below to make it move continously along a square path as shown in the video below:

  4. Update your code so that on starting the program, a check is performed to confirm whether the z-position of the sphere is greater than or equal to 75, if true then the sphere is moved using the MoveX() method until it's x-position is greater than or equal to 75. Note that you may have to use a nested if to implement this step and the three that follow.

  5. Update your code so that if the sphere's x-position is greater than or equal to 75, the sphere is moved along the z-axis using the MoveZ() method until it's z-position is less than or equal to -75.

  6. Update your code so that if the sphere's z-position is less than or equal to -75, the sphere is moved along the x-axis moved using the MoveX() method until it's x-position is less than or equal to -75.

  7. Update your code so that if the sphere's x-position is less than or equal to -75, the sphere is moved along the z-axis moved using the MoveZ() method until it's z-position is greater than or equal to 75.

  8. The sphere should now continously move in a square path.

  9. Can you make the sphere roll in the right direction while it moves? If you find this too complex, move to the next section and come back to this later.

2. Keyboard Input

  1. In your game-loop, add code to check for when the "W" key is hit. Whenever the key is hit, if the y-position of the sphere is less than 30 then it should be increased by 5.

  2. In your game-loop, add code to check for when the "S" key is hit. Whenever the key is hit, if the y-position of the sphere is greater than 10.0 then it should be decreased by 5.

  3. In your game-loop, add code to check for when the "D" key is hit. Whenever the key is hit, speed of the sphere should be increased by 10%.

  4. In your game-loop, add code to check for when the "A" key is hit. Whenever the key is hit, if the speed of the sphere is greater than 0 then it should be decreased by 10%.

3. Get Creative

  1. Think of more complex path shapes, and try making the sphere move along them.
